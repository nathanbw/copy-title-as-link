Copy Title As Link

This extension provides a keyboard shortcut (default: Alt-Shift-C / Option-Shift-C) that, when pressed, will copy a hyperlink to the currently-focused page into the clipboard. This link can then be pasted anywhere that accepts hyperlinks in text, like chat applications and office document applications.

There are also hotkeys to copy links as Markdown text (default: Alt-Shift-M / Option-Shift-M) or org-mode text (default: Alt-Shift-E / Option-Shift-E).

You can assign other hotkeys besides the defaults; navigate to about:addons and click the cog wheel in the top right and select "Manage Extension Shortcuts"
